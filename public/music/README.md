# Carpeta de Música

Coloca aquí tus archivos de música y letras:

## Archivos necesarios:

1. **song.mp3** (o song.ogg, song.wav, etc.)
   - Tu canción de fondo en formato de audio web compatible

2. **lyrics.lrc** (opcional pero recomendado)
   - Archivo de letras en formato LRC (tiempos sincronizados)
   - Formato:
     ```
     [mm:ss.xx]Texto de la línea
     [00:05.00]Primera línea de la canción
     [00:10.50]Segunda línea de la canción
     ```
   - mm = minutos, ss = segundos, xx = centésimas de segundo

## Ejemplo de lyrics.lrc:

```
[00:00.00]♪ Intro instrumental ♪
[00:05.50]Primera línea de la canción
[00:10.25]Segunda línea de la canción
[00:15.00]
[00:20.00]Tercera línea (línea vacía para pausas)
```

## Nota:

- La canción se reproducirá automáticamente al cargar la página
- Los subtítulos se mostrarán en una barra en la parte inferior
- Haz clic en la barra para ver todas las letras
- Si no hay lyrics.lrc, solo se mostrará el ícono de música sin texto
